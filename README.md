# PingyMaster
A Telegram bot for fast network diagnostics including ping, TCP port checks, IP lookup, and DNS queries.

PingyMaster is a Telegram bot that allows you to run quick and practical network checks without using a terminal, VPS access, or external tools.  
All checks are performed directly inside Telegram using simple commands.

## Telegram Bot
Bot Link: https://t.me/PingyMasterBot  
Bot Username: @PingyMasterBot

## Features
- Ping tests from multiple locations  
- TCP port checking  
- IP information lookup  
- DNS and dig queries  
- Service availability checks  
- Fast, lightweight, and easy to use  

## Technologies Used
PingyMaster is powered by reliable network and lookup services, including:
- Check-host
- IPinfo
- dig (DNS utilities)

No local installation or command-line access is required.

## Use Cases
- Quick server reachability checks  
- Port availability verification  
- IP inspection and geolocation lookup  
- DNS troubleshooting  
- Basic service status monitoring  

Suitable for DevOps engineers, developers, system administrators, and network enthusiasts.
